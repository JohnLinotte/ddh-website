Les essais

L'auteur réfléchit publiquement. Il prend position contre des sources nommées, il argumente, il tranche. Un essai paraît épisodiquement — un texte tenu, pas une cadence. Chaque essai est gratuit en lecture ; sa licence sépare l'œuvre intellectuelle de la trace qui l'a produite.

Un harness, ses sections

Contraindre le modèle, ou ne pas être un harness.

Où vit l'agentivité opérationnelle dans un système d'agents qui travaille ? La position est tranchée : dans le code qui orchestre, pas dans le modèle. Le mot harness y retrouve son sens premier.
