La Bride

Les garde-fous d'Aegis, au seuil de Claude Code — le plugin aegis-core. La pièce qui retient l'agent : ce qu'il écrit, ce qu'il lance, ce qu'il dit, ce qu'il garde. Déterministe — aucun modèle dans la boucle de sécurité.

Un agent, au seuil, fait des choses qu'on ne voit qu'après coup : il écrit une clé dans un fichier, lance une commande qu'on ne rejouera pas, se laisse souffler une consigne par un texte tiers, agit sans laisser de trace, puis oublie au tour suivant. La Bride retient chacun de ces gestes — à l'entrée, à la sortie, au fil de la session. Quinze hooks, en Python, sans jamais convoquer un modèle pour décider.

01souveraineté

Le secret ne transite pas

la retenue · détection inversée

Une clé, un jeton, un identifiant ne quitte jamais votre machine par l'agent — ni dans un fichier qu'il écrit, ni dans une phrase qu'il dit. Là où les scanners tolèrent ‹votre-clé-ici›, La Bride le refuse.

02agents

Chacun dans son couloir

structure · écriture · shell audité

Contraintes structurelles sur chaque agent, ouvriers interdits d'écrire dans le cœur, exécution shell auditée. Un agent ne déborde pas de son mandat.

03irréversible

Le destructeur est gelé

confirmation obligatoire

Toute action irréversible — suppression, écrasement, envoi — passe par une confirmation explicite. Rien de définitif ne se fait dans le dos de l'opérateur.

04injection

Le texte tiers ne pilote pas

entrée mémoire · sortie d'outil

Les entrées mémoire et les sorties d'outils sont assainies. Ce qu'un document ou une page web tente de glisser à l'agent n'en prend pas le contrôle.

05forensique

Rien qui ne soit consigné

trace opposable

Chaque appel d'outil — natif ou MCP — est enregistré, écrit atomiquement. Pas un journal de debug : une pièce qu'on peut produire devant un tiers.

06continuité

Le seuil se souvient

compaction · modèle mental

Sauvegarde et restauration du contexte autour de la compaction, complétude vérifiée avant l'arrêt, modèle mental mis à jour après chaque dispatch. La mémoire de travail survit au tour.

pourquoi au seuilLe garde-fou est à la passerelle, pas dans le code

Le consensus de 2026 est net : un garde-fou efficace ne vit pas dans le code de l'application — il est à la passerelle. La Bride s'installe au seuil de Claude Code : elle voit passer chaque écriture, chaque commande, chaque sortie, chaque fin de tour, et applique la même discipline partout, sans réécrire l'agent. Déterministe — aucun modèle dans la boucle de sécurité —, donc rejouable, auditable, sans coût de jeton.

Et la contrainte ne coûte pas la liberté. Elle porte sur l'irréversible — l'écriture, l'état, l'action — jamais sur le raisonnement. L'agent reste libre entre les rails : c'est précisément ce qui permet de faire tourner un opérateur autonome à pleine latitude sans rien céder sur ce qui ne se rejoue pas.

preuve à froidElle arrête la main qui code

Pendant son propre développement, La Bride a refusé une écriture qui contenait l'instruction « collez votre jeton ici ». Le refus, livré mot pour mot à l'agent : 

DENY · credential_scanner
(Credential placeholder / fill-me assignment)

The assistant must never be the conduit for secrets or
config values — not in files, not in chat, not in
prose / guides.

Safe pattern: user → the tool's config file (or an env
var) → the tool's code reads it at runtime. Identifiers
like account_id / zone_id belong in the config file too
— never relayed through the conversation.

périmètreCe que La Bride ne fait pas

Une bride retient ; elle ne fait pas le travail des autres pièces de l'attelage — et le dit.

  • Pas un coffre. La Bride ne stocke, ne chiffre ni ne fait tourner les secrets : elle agit à la source, pour qu'ils n'entrent pas dans le tuyau. Le coffre, la rotation, le chiffrement au repos sont le métier d'un gestionnaire de secrets, pas le sien.
  • Le garde de parole corrige, il ne mure pas. Sur la réponse de l'agent, il signale et fait reprendre avant la fin du tour ; il s'active au démarrage de session.
  • Les détecteurs attrapent des formes, pas des intentions — ils ont, comme tout détecteur, leurs angles morts. Les composants relèvent de techniques connues : la valeur tient à l'assemblage, à la détection inversée et au déterminisme de bout en bout, pas à une nouveauté brevetable.
