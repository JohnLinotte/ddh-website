L'agentivité ne vit pas dans le modèle. Elle vit dans le code qui l'orchestre.

Un harness route une force brute en travail dirigé. Le modèle lit, propose, reformule ; le code déterministe tranche, écrit, garde la trace. Le dernier mot n'appartient pas à la probabilité : il revient au code qui l'encadre.

la trace certifiée

Chaque pièce arrive avec son dossier zippé, anonymisé, vérifiable. On ne raconte pas la fabrication : on la pose à plat, à côté de l'œuvre.
