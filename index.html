section des harnais · texte-comme-image

L'agentivité ne vit pas dans le modèle.Elle vit dans le code qui l'orchestre.

Un harness route une force brute en travail dirigé. Le modèle lit, propose, reformule ; le code déterministe tranche, écrit, garde la trace. Cet atelier publie ce déplacement sous deux matières attachées — la spécification d'un harness, et les dossiers qui la démontrent en train de tenir.

<operator />
un humain qui répond

Les décisions de goût et d'arbitrage ne se délèguent pas à une probabilité. Elles tiennent à une signature qui répond en première personne.

<harness />
l'architecture déterministe

Test avant inférence, écritures atomiques, permissions stratifiées, gates par type de sortie. La contrainte est lisible, datable, opposable.

<artifact />
la trace certifiée

Chaque pièce arrive avec son dossier zippé, anonymisé, vérifiable. On ne raconte pas la fabrication : on la pose à plat, à côté de l'œuvre.
